# review-edit

プルリクエストに対してレビューコメントがありました。
以下の対応をお願いします。

## お願いしたいこと

1. **レビュー内容の解説**: 各指摘について、何が問題とされているのか、なぜその指摘がされたのかを解説してください
2. **対応方針の提案**: 指摘ごとに対応方針（修正する / 反論する / 議論が必要）を提案してください
3. **修正案の提示**: 修正が必要な場合は、具体的なコード修正案も提示してください

## 入力

以下の情報が入力されるので、それを使ってください。

- **PR URL**: プルリクエストのURL（1つ）
- **レビューコメント URL**: レビューコメントのURL（複数可）

また、レビュー内容のテキストや @ファイル名 で対象ファイルが渡される場合もあります。

## 出力形式

指摘ごとに以下の形式でまとめてください:

### 指摘1: （指摘の要約）
- **解説**: ...
- **対応方針**: 修正する / 反論する / 議論が必要
- **修正案**: （必要な場合）

### まとめ（複数指摘がある場合）

| # | 指摘の要約 | 対応方針 |
|---|-----------|---------|
| 1 | ... | 修正 / 反論 / 議論 |
| 2 | ... | 修正 / 反論 / 議論 |
