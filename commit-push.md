# コミット&プッシュ

変更をコミットし、GitHub CLIを用いてpushしてください。

- コミットメッセージは日本語で作成すること。
- 読みやすいように箇条書きなどを活用すること。
- コミット対象がなければ、pushのみ行うこと。
- worktree内で作業中の場合は、それも考慮すること。

## 改行について

- `git commit -m "…\n…"` のように、`-m` に渡す文字列の中で `\n` を書いても、実際の改行にはならず、`\n` という文字列がそのまま表示されてしまいます。
- そのためコミットメッセージは、「実際の改行」を含む形で渡す（\n を文字として入れない）
